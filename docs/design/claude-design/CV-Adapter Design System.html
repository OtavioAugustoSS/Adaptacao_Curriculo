cv CV-Adapter · design system

Proposta para aprovação · v0.1

Um sistema de design dev-tool para o CV-Adapter.

Limpo, alto contraste e tipograficamente preciso — pensado para uma ferramenta de produtividade, não para um “template de IA genérico”. Tokens em light + dark, prontos para mapear em Tailwind no handoff Next.js. Use o toggle acima para ver tudo nos dois temas.

Cor

Neutros frios como base, um azul confiante como único acento. Os cinzas carregam a interface; o acento é reservado para ação e foco. Cada token tem par em light e dark.

Acento azul · base #2f63ff (light) / #4f7cff (dark). Refinamento do #1a5cff original — mais frio e legível sobre branco e sobre fundo escuro. --accent

Acento usado com parcimônia: botão primário, foco, estado ativo da nav, links e o destaque do modo “Adaptar à vaga”. Tudo o mais é neutro. Se preferir outro acento (ex.: verde-esmeralda ou violeta Linear), troco só esta variável.
