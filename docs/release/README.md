# Relatórios de Release

Relatórios consolidados emitidos pelo `lead-agent` ao fim de cada fatia/versão
(ex.: `0_1_0.md`). Cada relatório cobre: funcionalidades implementadas (US + status),
dúvidas funcionais/técnicas levantadas, riscos mapeados, pendências, versão executiva e
próximo passo. Ver `template-claude-workflow.md` §6.2 (Lead) e §6.6 (checkpoints).
